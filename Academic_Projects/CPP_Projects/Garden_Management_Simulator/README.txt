This project is developed by using C++ programming logics.
This is Garden management Simulator for farmers.

When you run this program with a3main file, you can see this menu.

--------------------------------------------------------------------------
Please select from the following options :
 N - Create New Plant Type
 P - View Plants Type
 A - Create a new Crop Shape
 C - View Crop(s)
 D - Create Detailed Garden
 G - Add Crop to Garden
 V - View Garden
 L - Load Garden from a file
 S - Save Garden to a file
 Q - Quit
 Your selection :
